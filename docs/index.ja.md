# Spikuit

**ニューラルナレッジグラフ x 間隔反復**

FSRS x ナレッジグラフ x 活性化拡散 x 会話型RAG

---

Spikuit（spike + circuit、発音: /spai.kit/）は、記憶を神経回路としてモデル化するナレッジシステムです。ある概念を復習すると、つながった知識へ活性化が伝播し、使う経路は強化され、使わない経路は自然に減衰します。

**人間の学習ツール**としても、**AIエージェントのRAGブレイン**としても機能します。学習を助けるグラフが、そのままインテリジェントな検索エンジンにもなります。

## 主な特徴

- **FSRSスケジューリング** -- ニューロン単位のエビデンスベース間隔反復
- **グラフ伝播** -- 1つの概念を復習すると近傍が活性化（APPNP + STDP + LIF）
- **意味検索** -- sqlite-vecによる埋め込みベースの検索（プロバイダー差し替え可能）
- **自己最適化RAG** -- 会話のフィードバックで検索品質が向上
- **会話型キュレーション** -- 対話を通じてナレッジグラフを構築・改善
- **スキャフォールド適応学習** -- FSRS状態 + グラフ近傍からZPD着想のサポートレベル
- **プロジェクトローカルBrain** -- `.spikuit/` で `.git/` のようなCLI探索

## ニューロンモデル

| 脳 | Spikuit | 役割 |
|----|---------|------|
| ニューロン | `Neuron` | 知識の単位（Markdown） |
| シナプス | `Synapse` | 型付き・重み付きの接続 |
| スパイク | `Spike` | 復習イベント（活動電位） |
| 回路 | `Circuit` | ナレッジグラフ全体 |
| 可塑性 | `Plasticity` | チューニング可能な学習パラメータ |
